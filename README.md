# Weather Dashboard

## 📌 Description

A real-time Weather Dashboard web application built using HTML, CSS, JavaScript, and OpenWeather API.

This project allows users to search for weather information by entering a city name. It displays real-time weather details such as temperature, humidity, weather condition, and wind speed.

---

## ✨ Features

- Search weather by city name
- Display real-time temperature
- Display humidity
- Display weather condition
- Display wind speed
- Uses OpenWeather API
- Simple and clean user interface

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API
- Fetch API

---

## 🎯 Purpose of the Project

This project was created to practice frontend development skills and learn how to work with APIs in JavaScript.

Through this project, I learned how to fetch real-time data from an external API and display it dynamically on a webpage.

---

## 📚 What I Learned

- How to use API in JavaScript
- How to use fetch()
- How to use async and await
- How to display API data on a webpage
- How to handle user input
- How to show weather details dynamically

---

## 🚀 How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open index.html in any web browser
4. Enter a city name
5. Click the Search button to view weather details
